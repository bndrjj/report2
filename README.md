# 📊 نظام تقارير دعم التميز المدرسي — GitHub Pages

نظام ويب مجاني يعمل مباشرةً من المتصفح لإنشاء تقارير PDF آلية من بيانات Excel (Microsoft Forms).

---

## 🚀 خطوات الإعداد (مرة واحدة فقط)

### الخطوة 1 — إنشاء Repository على GitHub

1. افتح [github.com](https://github.com) وسجّل دخولك
2. اضغط زر **"New"** أو **"+"** لإنشاء repository جديد
3. اسمِه: `tamaruz-reports` (أو أي اسم تختاره)
4. اختر **Public** (ضروري لتفعيل GitHub Pages مجانًا)
5. اضغط **"Create repository"**

### الخطوة 2 — رفع ملف index.html

1. في صفحة الـ repository الجديد، اضغط **"uploading an existing file"**
2. ارفع ملف `index.html` (الملف الذي أُنشئ لك)
3. اضغط **"Commit changes"**

### الخطوة 3 — إنشاء مجلد data ورفع ملف Excel

1. في الـ repository، اضغط **"Add file"** → **"Create new file"**
2. في حقل الاسم اكتب: `data/report.xlsx` ثم أضف أي حرف مؤقت واحذفه
3. اضغط **"Commit changes"**
4. ثم ارفع ملف Excel باسم `report.xlsx` داخل مجلد `data/`

> **ملاحظة:** يمكنك أيضًا رفع الملف مباشرةً من الموقع بدون حفظه في الـ repo.

### الخطوة 4 — تفعيل GitHub Pages

1. اضغط على **Settings** في أعلى صفحة الـ repository
2. من القائمة الجانبية اختر **Pages**
3. في قسم **Source** اختر: **Deploy from a branch**
4. في **Branch** اختر: **main** ثم **/ (root)**
5. اضغط **Save**
6. انتظر دقيقتين، ثم ستجد رابط موقعك مثل:
   ```
   https://اسمك.github.io/tamaruz-reports/
   ```

## 🔄 التحديث الأسبوعي للبيانات

### الطريقة الأسرع — رفع من الموقع مباشرةً:
1. افتح الموقع من رابط GitHub Pages
2. اسحب ملف Excel الجديد وأفلته في منطقة الرفع
3. ✅ الأمر انتهى — لا تحتاج GitHub على الإطلاق!

### طريقة بديلة — استبدال الملف في GitHub:
1. في GitHub، انتقل إلى `data/report.xlsx`
2. اضغط على أيقونة القلم (Edit) أو زر رفع ملف جديد
3. ارفع ملف Excel الجديد بنفس الاسم `report.xlsx`
4. اضغط **"Commit changes"**
5. ✅ سيتحدث الموقع تلقائيًا خلال دقيقة
